Student Management & B2C E-Commerce Web Application
📌 Overview

This project is a web-based ASP.NET Core MVC application developed for academic purposes. It integrates a Student Management System and a Business-to-Consumer (B2C) E-Commerce platform into a single, cohesive web application.

The system demonstrates core web development concepts including authentication, role-based access control, CRUD operations, and responsive UI design.

🚀 Technologies Used
ASP.NET Core MVC
C#
Entity Framework Core
SQL Server
HTML5 & CSS3
Bootstrap
🔐 Authentication & Roles

The application implements built-in ASP.NET Core authentication with role-based access control:

👤 Student
Register and log in
View and update personal information
🛍️ Consumer
Register and log in
Manage personal profile details
🛠️ Admin
View all users (students and consumers)
Delete user accounts
🎓 Student Management System

The Student Management System serves as the landing component of the application.

Key Features
Student registration with authentication
Create, view, update, and delete student records
Searchable student list
Profile image upload and management
🛒 B2C E-Commerce Platform
🌐 Website Features
Structured layout with header and footer
About page describing the business
Contact form for user inquiries
Social media integration
👥 Consumer Management
Create, view, update, and delete consumer records
Searchable consumer list
Profile image upload functionality
🎯 Additional Features

Displays relevant images for products/services
User-controlled (pause/stop)
Integrated with responsive design
📢 Promotions & Advertisements
